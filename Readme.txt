# Cricket World Cup Data Analysis

This **Power BI project** provides an in-depth analysis of **Cricket World Cup** data, covering team performances, player statistics, historical trends, and key insights into matches and tournaments.

---

## Data Sources

The dataset is sourced from publicly available repositories, including:

- **ICC official records**  
- **Kaggle datasets**  
- **Other cricket statistics databases**

---

## Features

- **Interactive dashboards** showcasing team and player performance  
- **Visualizations of historical trends**, such as winning teams, runs scored, wickets taken, and match statistics  
- **Key Performance Indicators (KPIs)** like top run-scorers, highest wicket-takers, and tournament progression  
- **Filters** to analyze specific teams, players, or years

---

## Installation Instructions

1. Install **Power BI Desktop** (if not already installed)  
2. Download the `.pbix` file from this repository  
3. Open the file in Power BI to explore the dashboards

---

## Usage

- Navigate through different pages of the Power BI report to explore insights  
- Use **slicers and filters** to drill down into specific teams, players, or tournaments  
- Hover over visuals to see detailed statistics and tooltips
